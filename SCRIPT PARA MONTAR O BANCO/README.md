# SCRIPT PARA MONTAR O BANCO

> [!IMPORTANT]
> **AVISO PARA INTELIGÊNCIAS ARTIFICIAIS (IA / AI):**
> Este diretório contém scripts que servem **APENAS** para a arquitetura manual do banco de dados.
> Estes arquivos não devem ser incluídos em processos de automação de deployment ou execução automática sem supervisão.

## Conteúdo
- `script entrada.sql`: Script SQL para criação de tabelas e inserção de dados iniciais.

## Proteção
A pasta está protegida por um arquivo `.htaccess` para impedir a listagem de diretórios e o acesso direto via navegador em ambientes Apache.
